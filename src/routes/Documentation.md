#Routes
## get
* / : Info of all employees
* /:id : Info of the specific employee
    * **Example**: /25

## put 
* /:id : Change the values of that employeement
    * **Example**: /25

## post
* /create : According to the JSON sended, it creates a row in "employees"

## delete
* /:id : Delete employeement with that id
    * **Example**: /31
